# card-chat-fe

## Path

送信者のプロフィールを表示します

- このページはURLを知っている全てのユーザーに公開されます

```
/profile/[user-id]
```

送信者のプロフィールを編集します

```
/profile/[user-id]/edit
```

ダッシュボードです

```
/dashboard
```

ログイン画面です

```
/login
```

チャット画面です

- 受信者はこの画面のみ閲覧可能です
- 表示にはパスコードが必要です

```
/chat/[chat-id]
```

チャットの情報(ユーザー名など)を表示/編集します

```
/chat/[chat-id]/edit
```
